Benefits of Font Icons

  • The size, color, shadows can be controlled via CSS
  • The opacity and rotation can be controlled via CSS
  • You can add strokes to them with text-stroke or add gradients/textures with background-clip: text; once browser support is better.
  • They are vector graphics so they are scalable
  • They load fast
  • They have good browser support

Drawbacks

  • The icons are anti-aliased, as the browser see the icons as text, so the icons may not be very sharp
  • They are usually limited to one colour
